============
Installation
============


Stable release
--------------

To install kg_toxic_comment_classification_challenge, run this command in your terminal:

.. code-block:: console

    $ pip install kg_toxic_comment_classification_challenge

This is the preferred method to install kg_toxic_comment_classification_challenge, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for kg_toxic_comment_classification_challenge can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/BlackPoint_CX/kg_toxic_comment_classification_challenge

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/BlackPoint_CX/kg_toxic_comment_classification_challenge/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/BlackPoint_CX/kg_toxic_comment_classification_challenge
.. _tarball: https://github.com/BlackPoint_CX/kg_toxic_comment_classification_challenge/tarball/master
